각 서버는 오라클 클라우드의 가상머신으로 구축되었습니다.
- **서버 사양**
	- **운영체제** : Canonical Ubuntu 22.04
	- **CPU** : VM.Standard.E2.1.Micro 1OCPU(2vCPU)
	- **메모리** : 1GB
	- **볼륨** : 50GB
	- **대역폭** : 0.48Gbps
	- **트래픽 제한**: 10TB/월
## 서버 목록
- **데이터베이스 서버**  
	- **DBMS** : PostgreSQL
	- **버전** : 16
	- **용도** : 애플리케이션 서버에서 사용할 데이터 관리
- **애플리케이션 서버**  
	- **용도** : 개발된 프로그램들의 실행

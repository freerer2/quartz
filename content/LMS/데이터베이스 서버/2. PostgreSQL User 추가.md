# 유저 추가
`superuser`로 진행
```sh
# 유저추가
sudo adduser test

# 데이터베이스 유저 추가
sudo -i -u postgres
createuser --interactive

Enter name of role to add: test
Shall the new role be a superuser? (y/n)

# 데이터베이스 유저 디폴트 데이터베이스 추가
createdb test
```
## 비밀번호 초기화
`superuser`로 진행
```sh
sudo passwd test
sudo -i -u test
psql

test=# \conninfo
test=# ALTER USER test PASSWORD 'myPassword';
```

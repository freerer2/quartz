Java 프로그래밍 언어는 예외를 사용하여 오류 및 기타 예외적인 이벤트를 처리합니다. 이 튜토리얼에서는 예외를 사용하는 시기와 방법에 대해 설명합니다.


1. [[예외란]]  
	예외가 무엇인지 소개합니다.
	
2. [[예외 감지 및 처리]]  
	try, catch, finally를 사용하는 방법에 대해 설명합니다.
	
3. [[예외 던지기]]  
	프로그램에서 예외를 던지는 방법
	
4. [[확인되지 않은 예외]] — 논란
	확인 또는 확인되지 않은 예외를 사용해야 하는 경우

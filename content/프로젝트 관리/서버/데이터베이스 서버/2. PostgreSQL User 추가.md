# 유저 추가
`superuser`로 진행
```sh
# 유저추가
sudo adduser [유저명] #ex) gitea

# 데이터베이스 유저 추가
sudo -i -u postgres
createuser --interactive

Enter name of role to add: [유저명] #ex) gitea
Shall the new role be a superuser? (y/n)

# 데이터베이스 유저 디폴트 데이터베이스 추가(유저명 권장)
createdb [db명]

# 신규 추가된 DB유저 비밀번호 초기화
psql
postgres=# ALTER USER 유저명 PASSWORD 'myPassword';

# 접속 테스트
su [유저명] #ex) gitea
psql
gitea=# \conninfo
gitea=# \q
```

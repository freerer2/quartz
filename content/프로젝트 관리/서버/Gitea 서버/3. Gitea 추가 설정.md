## 이메일 설정

```sh
sudo vi /etc/gitea/app.ini
```

```ini
;지메일 사용시
;2FA 활성화 할 것
[mailer]  
ENABLED = true  
HOST = smtp.gmail.com:465 ; Remove this line for Gitea >= 1.18.0  
SMTP_ADDR = smtp.gmail.com  
SMTP_PORT = 465  
FROM = "name"<example.user@gmail.com>
USER = example.user
PASSWD = `**** **** **** ****`  
PROTOCOL = smtps
```

## 기타 설정

```sh
sudo vi /etc/gitea/app.ini
```

```ini
[service] 
DISABLE_REGISTRATION = true ; 신규가입 비활성
REQUIRE_SIGNIN_VIEW = true ; 로그인 한 회원반 조회 가능

[i18n]
LANGS = ko-KR,en-US ; 한글설정
NAMES = 한국어,English ; 언어명
```

## 서비스 재시작

```sh
sudo systemctl restart gitea
```
